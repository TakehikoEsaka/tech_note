# コマンド
* which (コマンド名)でそのコマンドがどこにあるかわかる。
* 例えばdocker toolboxをwindowsにインストールしたとき、which dockerとしたときの出力は”/c/Program Files/Docker Toolbox/docker"

結果的に、python = 3.6.8 / tensorflow-gpu =　1.12 / cuda = 9.0 / cudnn = 7.4
